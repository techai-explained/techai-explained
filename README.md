## 🎯 Mission

TechAI Explained breaks down complex AI, DevOps, and software engineering topics into digestible 5-10 minute videos. Fully automated production pipeline — from script to upload.

## 🎥 Content Pipeline

```
Topic Idea → Script (Squad) → Voice (edge-tts/Azure) → Visuals (screen rec + diagrams) → Edit (ffmpeg) → Upload (YouTube API)
```

## 🛠️ Tech Stack

- **Voice:** edge-tts (free) → Azure Neural Voices (premium)
- **Visuals:** OBS for screen recording, Excalidraw for diagrams
- **Editing:** ffmpeg + Python automation
- **Upload:** YouTube Data API v3
- **Scripts:** Generated by Squad agents
