☸️ Kubernetes for Application Developers

A 5-lesson mini course. No DevOps background required — just a developer who wants to understand where their code actually runs.

## Course Overview This course teaches Kubernetes from the perspective of an **application developer**, not a cluster administrator. You'll learn the resources that directly affect your code — Pods, Deployments, Services, ConfigMaps, and Secrets — with practical YAML examples you can copy into your projects. **Prerequisites:** Basic Docker knowledge (building images, running containers). No Kubernetes experience needed. ## Lessons
Lesson 1

Pods, Deployments, Services — The Core Three

The three resources every developer must understand. How they relate, when to use each, and complete YAML examples.

~15 min read
Lesson 2

ConfigMaps, Secrets, and Environment Management

How to configure your applications in Kubernetes — environment variables, config files, secrets, and the patterns that work in production.

~15 min read
Lesson 3

Persistent Storage — PVs, PVCs, and StorageClasses

How Kubernetes handles persistent data — PersistentVolumes, PersistentVolumeClaims, StorageClasses, and StatefulSets.

~15 min read
Lesson 4

Networking — Ingress, NetworkPolicies, and Service Mesh Basics

How traffic reaches your apps — Ingress controllers, TLS certificates, internal firewalls, and service mesh concepts.

~15 min read
Lesson 5

Production Readiness — Health Checks, Resource Limits, and PDBs

The final checklist — readiness probes, liveness probes, resource requests/limits, Pod Disruption Budgets, and graceful shutdown.

~15 min read
