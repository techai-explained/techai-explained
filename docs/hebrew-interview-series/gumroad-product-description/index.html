📚 מה תקבלו?

🔵 מסלול 1: C# & .NET (10 שיעורים)

הנושאים שנשאלים בכל ראיון .NET:

✅ Value Types vs Reference Types — ההבדל שכולם שואלים, עם המלכודות שרוב המועמדים נופלים בהם
✅ async/await לעומק — מה באמת קורה כשאתם כותבים await, למה deadlocks קורים ואיך למנוע
✅ Garbage Collector — Generations, IDisposable, ולמה GC.Collect() כמעט תמיד רעה
✅ Dependency Injection — Transient/Scoped/Singleton, Captive Dependencies
✅ LINQ וביצועים — Deferred Execution, IEnumerable vs IQueryable, N+1
✅ Interfaces vs Abstract Classes — מתי כל אחד ולמה
✅ Generics ו-Constraints — Covariance, Contravariance, out/in
✅ Threading ו-Concurrency — Race Conditions, lock, SemaphoreSlim, CancellationToken
✅ Design Patterns — Singleton, Factory, Repository בC# המודרנית
✅ Entity Framework Core — Migrations, N+1, Eager/Lazy Loading, AsNoTracking

🟢 מסלול 2: Algorithms & LeetCode (10 שיעורים)

הנושאים שחוזרים בכל ראיון אלגוריתמי:

✅ Big O Notation — כיצד לחשב ולהסביר סיבוכיות תוך שניות
✅ Arrays ו-Strings — Two-Pointer, Sliding Window, שאלות נפוצות
✅ Linked Lists — Reversal, Fast & Slow Pointers, Cycle Detection
✅ Binary Search — ויריאנטים, Rotated Array, Off-by-one errors
✅ Stacks ו-Queues — Monotonic Stack, Min Stack, Parentheses
✅ Trees ו-BFS/DFS — InOrder/PreOrder, Level Order, LCA, Validate BST
✅ Dynamic Programming — Memoization vs Tabulation, Knapsack, Coin Change
✅ Graphs — BFS, DFS, Topological Sort, Dijkstra
✅ Sorting — Merge Sort, Quick Sort, מתי כל אחד
✅ Sliding Window — Fixed/Variable, Templates שאפשר להכיל
